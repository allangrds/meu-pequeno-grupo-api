## Requisitos

- Docker v19.03.5
- Docker Compose v1.21.2

## Instalação

Execute o seguindo comando no seu terminal:

```bash
cp .env.example .env
make bootstrap
```

Pronto! O projeto estará funcionando no endereço `http://localhost:<.env.API_PORT>`

## Documentação

Execute o seguindo comando no seu terminal:

```bash
npm run docs:preview
```
